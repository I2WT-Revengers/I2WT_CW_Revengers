How to create a website








Usually, three fundamental technologies are used for development:

HTML (from English HyperText Markup Language) is a hypertext markup language for a page. It is used to tell the browser how to display the loaded site. The language consists of tags - these are a kind of commands that are converted into visual objects in the user's browser. For example, the img tag is used to position images on a page. It has the required src attribute, which specifies a link to the file. This is what the code will look like:

<p> Some accompanying text </p> <img src = 'image.png'> <p> And here's another paragraph to fill the void. </p>
