How to create a website








1. Register a domain

Domain is the name of your site. It should be unique and convey the brand of your business. The easiest way to find and register a domain is to go to a domain registrar. A domain registrar will allow you to register your domain name under either annual contracts or long term contracts.

Advice:

  • If you find it difficult to figure out a domain name, you can always use domain name generators for ideas. Some of the most popular are Wordroid and Lean Domain Search.

  • If you're a beginner, we recommend going to Namecheap to find and buy a domain name.

  • + read more about domains

2. Buy web hosting

A web host is a large computer (aka a server) that hosts your websites. Some giant companies like Amazon, IBM and FB own and operate their own web servers; other companies simply rent their servers from a hosting provider (which is much cheaper and easier).

Today, web hosting comes in a variety of packages.

Traditionally, a web hosting provider is only concerned with renting out servers with basic software and technical support. Hosting your website with a traditional hosting provider is usually cheaper and more flexible; but you need a little extra work to set up the website.

Companies are now bundling various services together and allowing users to create, host and manage websites from a single service provider. These companies are commonly referred to as website builders or online store builders. Hosting a website on these "complex" platforms is usually expensive and less flexible; but you can easily create a website or online store.

Advice:

  • There are different types of hosting available: shared hosting, dedicated server hosting and cloud / VPS hosting.

  • If you have a small website, it is cheaper to choose a shared hosting plan. For large sites, it is recommended to use cloud or dedicated hosting.

  • Some key qualities to consider before choosing a web host are their price, server speed and uptime.

3. Create your web page

Once you've got your domain name and web hosting, it's time to roll up your sleeves and get creative! To design and build web pages, you can hire a web developer or do it yourself.

You can create your own unique and unique website yourself if you know web languages and website basics.

HTML (from English HyperText Markup Language) is a hypertext markup language for a page. It is used to tell the browser how to display the loaded site. The language consists of tags - these are a kind of commands that are converted into visual objects in the user's browser. For example, the img tag is used to position images on a page. It has the required src attribute, which specifies a link to the file.
This is what the code will look like:

<p> Some accompanying text </p> <img src = 'image.png'> <p> And here's another paragraph to fill the void. </p>

Example of code:

<tagname>{
attribute1(i.e color):black;
attribute2(i.e width):5px;
}

IDE

A good local working environment is essential for creating and organizing website source files. You can create a web development environment on your computer device by installing an IDE (Integrated Development Environment). The IDE mainly consists of a text editor, build automation, and a debugger. Sublime Text and Brackets are some of the main IDEs for web development, supporting HTML, CSS, JS, PHP, Python and similar web languages. On the other hand, there are advanced IDEs such as Adobe Dreamweaver that offer several other features (eg server connection, FTP).
