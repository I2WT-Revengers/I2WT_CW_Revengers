History









Historical data 1

  • CSS was first proposed in 1994 by Håkon Wium Lie, associate of Tim Berners-Lee at CERN
  • After a few discussions, the first W3C emerged in 1996 Recommendation for CSS (CSS1).
  • CSS allows the style of a document to be influenced from multiple style sheets (“cascading”). Some of the Features provided by the CSS1 specification were:
    • Supporting font properties
    • Specifying font color, background and more elements
    • Arranging font alignment, images, tables, etc.

Historical data 2

  • That's why the W3C started the review of the CSS2 specification and its creation CSS specification 2.1
  • The creation of the CSS3 specification began in 1998 and is under development
  • While CSS2 is a great specification which defines various features, CSS3 is divided into many separate files ("modules")
  • Each of these modules adds or extends features already in CSS2, thus maintaining "backwards" compatibility (backwards compatibility)

Historical Table

Version Characteristics
CSS 1 The W3C was formally recommended in December 1996 used in most browsers since 2000. Among other things, he argued:
  • Align text, images, tables, etc.
  • Font properties
  • Unique ids for data
  • Classes for property groups
CSS 2 Published as W3C Recommendation in May 1998, but never supported fully.Some new features he introduced:
  • Media types
  • z-index
  • Bidirectional text
  • New properties for fonts (eg shadow)
CSS 2.1 Published in June 2005.
  • Fixed errors of the CSS2 specification
  • Removes items that are not fully supported by browsers
  • Extends fully supported features
CSS 3.0 Under development. Basic features:
  • It is seperated into modules.
  • It consists of many different specifications
  • Introduces new features and effects, such as: animations, transitions, gradients, multiple backgrounds, 3D transforms, etc.
  • Provides the ability for advanced selectors
