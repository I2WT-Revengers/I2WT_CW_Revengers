What is Cascading Style Sheets?

Part1

  • CSS stands for Cascading Style Sheets
  • Each HTML element has a predefined style
  • Changing the default style of an HTML element is done via the style attribute
  • This way of changing the style, through its use style attribute of HTML elements- is called inline
  • There are two other ways to change the style:
    • the internal way - using,<style> element in the <head> part of the HTML document, where the desired styles of HTML document elements are.
    • or in the external way - using,that is, one or more external Style Sheets which stored in CSS files

Part2

  • The most common and the suggested way for change of the style of a web page
  • Simple style sheet language that describes appearance and formatting of a document which has been written using sign language
  • Its most common use is to change style web pages written in HTML
  • Along with HTML and JavaScript, CSS is basic technology used extensively today for creating websites and interfaces user for web and mobile applications

Part3

  • Designed primarily to facilitate separation the content of the document from its presentation, for reasons of accessibility, flexibility, easier sharing and less complexity due to minimizing code duplication in its structure content
  • Allows the designer / developer to separate instructions on how they will appear the various items on the website in a separate file or in a section of the HTML document
  • In that way, keeping the style instructions in external .css files, can be changed very easily showing multiple websites, saving effort and time

Part4

  • For this reason, the World Wide Web Consortium (W3C) created CSS, with the goal the separation of all formatting from HTML document, and save it to a separate CSS file. Therefore, CSS:
    • Specifies the presentation level layer of the user interface
    • Defines to the browser how each item is displayed on the users screen
