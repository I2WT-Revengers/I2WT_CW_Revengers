How to Link HTML file to CSS





External CSS

In case that we want a more complicated process, such as changing the font or background of the page,we would create a separate CSS file for this purpose, describing the Formatting of the different sections of the page, as follows:

  • Create a new CSS file (eg template.css) with the desired code formatting content.
  • Link the formatting content to the HTML file.

Example.

External styles are defined within the <link> element, inside the <head> section of an HTML page:

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="mystyle.css">
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
</body>
</html>

An external style sheet can be written in any text editor, and must be saved with a .css extension.

The external .css file should not contain any HTML tags.

Here is how the "mystyle.css" file looks:

body {
background-color: lightblue;
}

h1 {
color: navy;
margin-left: 20px;
}

Note that to achieve the above connection, these two files will must be on the same path (folder)

This way, however, is not the only one. Two other methods exist that someone could achieve the same same results and are described below, but they are used only in small sites, due to the increasing of complexity of the code involving:

Inline CSS

In case that the formatting concerns only one (or generally few) item and is not intended to be reused, it may integrate it into an attribute style.Given this, it is reasonable to ask what will happen when formatting an element is defined by both an attribute style in the HTML file, as well as from a separate CSS file.In this case, the browser will prioritize formatting within the attribute style for this item.

Example

Inline styles are defined within the "style" attribute of the relevant element:

<!DOCTYPE html>
<html>
<body>
<h1 style="color:blue;text-align:center;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>
</body>
</html>
Internal CSS

There are cases that we would like to apply some CSS rules only to a specific page and not to the whole site.
An internal style sheet may be used if one single HTML page has a unique style.
The internal style is defined inside the <style> element, inside the head section.

Examle

<!DOCTYPE html>
<html>
<head>
<style>
body{
background-color: linen;
}
h1 {
color: maroon;
margin-left: 40px;
}
</style>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
</body>
</html>
