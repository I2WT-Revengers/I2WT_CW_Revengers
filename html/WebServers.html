Web Servers








What are servers?

To answer this question, you need to understand where the word "server" comes from. It is based on the English verb to serve, which translates as "to serve." So, server, in simple words, is a special computer that serves to ensure that this or that information network (for example, local) can fully function.

Server[1]

Functions:

Machine, depending on its type and purpose, can perform various useful functions:

  • Store information from one or more sites. This is how the servers of Internet providers providing hosting services work;
  • Coordinate the interaction of many computers on the same network. A classic example is game servers;
  • Used to store corporate data and employee access to them. One possible example is a server at the headquarters of a company that stores and regularly updates an accounting database.

Hardware

Any server, regardless of its type and purpose, is a computer. That is why he has on board classic computer components:

  • The motherboard, which plays the role of the basis of the system;
  • One or more central processing units;
  • Certain amount of RAM
  • Data storage system consisting of drives of one type or another.

Server hardware[1]

The "hardware" of the server, while being fundamentally similar to the components of an ordinary personal computer, has its own specifics. It is connected with the fact that the main tasks of server equipment are storage, intensive processing and fast transfer of large amounts of data. For them to succeed, the servers equip:

  • High-performance multi-core processors;
  • Large amount of RAM with error control;
  • Capacious and high-speed hard drives and solid-state drives.

Software and types of Servers

Particular attention should be paid to the server software, which runs the above hardware. Ordinary operating systems - the same Windows, perhaps known to everyone - in this case are poorly suited or not at all. Server versions of the OS are used - both developed by Microsoft and included in the Linux family. The latter, by the way, have received the widest possible distribution, since they ensure the stability of servers, data security and configuration flexibility. Working with a server OS of the Linux family (for example, Ubuntu Server) requires special knowledge, so data centers and companies hire qualified IT specialists.If we are talking about a small server that does not require special functionality and processing power, an ordinary operating system may also work. Thus, the capabilities of Windows are more than enough to configure FTP and organize the operation of a file server accessible to a limited number of people.

Types:

  • A file server is a centralized storage of information, access to the disks of which have personal computers connected to the local network. The main task of the file server is reduced to reliable data storage and uninterrupted access to it, and in case of damage to files - their complete recovery.
  • The database server is not so much a means of storing and accessing, but of processing arrays of information. Through client requests, the requested information is retrieved, the data is processed, structured, and changed depending on the server settings. These servers are managed by DBMS (Database Management Systems), the most famous of them are MS SQL Server, Oracle, MySQL. The number of users and the size of the database, as well as the prospects for future growth, determine such important characteristics of the database server as capacity and scalability.
  • A print server (print server) allows one printing device to be used to service multiple computers. The function of the print server is to accept requests for print output, line them up and send them to the printer according to it. Thus, money is saved on equipping each computer with its own printer, their memory is freed up for other tasks, office space is rationally used.
  • A workgroup server is a multifunctional hardware solution for a group of computers (usually no more than 20). Combines the capabilities of a file server, application server, database, print / fax server, mail and others, depending on the needs. In general use, the workgroup server is obliged to delimit access to data and user rights. Usually has one processor, most often used in small firms where there is no need to allocate servers for separate tasks.
  • Domain Controller (Domain Controller server) - the main computer in a local network with a hierarchical structure - the domain. The domain controller is used to centrally manage domain resources â€” computer and user accounts. Using the Active Directory directory service, it stores user data and authenticates them to access local network resources. Works under control of server OS from MS Windows, starting from Windows 2000 Server. A domain controller is an important element of the network infrastructure of large companies. In addition, it can act as a file server and print server.
  • Mail server, or e-mail server, message server - the name speaks for itself. The main task of such a server is to recognize the addresses of incoming e-mail and distribute it to the intranet mailboxes, as well as send outgoing messages, and ensure internal correspondence. The mail server provides reliable filtering of spam and malware in messages and protects internal information from unwanted access.
  • FTP servers are an integral part of the technical support of the World Wide Web. Their task is to transfer files at the request of simple file managers using the standard File Transfer Protocol. The most advanced FTP servers are able to categorize files by type and location, restrict access to them, or provide opportunities for sharing on the Internet.
  • A proxy server is an intermediary between local network users and the Internet. Provides secure Internet access, protecting against unwanted access from outside and, if necessary, restricting access to certain resources for local network users. In addition, it performs a number of other functions: accounting and saving traffic by compressing data, caching, anonymizing access.
  • Web server (web application server) - a specially dedicated computer that is responsible for accessing the campaign site of Internet users, correct and fast display of static or dynamic pages. The web server is obliged to ensure the uninterrupted operation of the Internet resource, taking into account the attendance, resist network attacks, and prevent the possibility of hacking. The more important a website plays in a business process (for example, it provides communication with customers, is a sales channel for products), the more important this server is for it. In recent years, a web server is often called not the machine itself, but the program that performs the above functions.

Web servers more close:

A web server is a server that accepts HTTP requests from clients, usually web browsers, and issues HTTP responses to them, usually along with an HTML page, image, file, media stream, or other data. A web server refers to both the software that performs the functions of a web server and the computer itself on which this software runs.
